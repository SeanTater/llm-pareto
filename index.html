Estimation notes

We estimate missing pricing and parameters from a single anchor model: DeepSeek-R1 on OpenRouter. We assume the same input/output ratio and scale costs linearly with parameters.

Active-parameter estimate uses $0.300 input and $1.200 output per 1M tokens divided by 37B active parameters.

Total-parameter estimate uses the same prices divided by 671B total parameters.
